# Popular Movies App
App that shows most popular and top rated movies from the web [TMDb](https://www.themoviedb.org)

## Getting Started

To get the project working we need and API Key from the [TMDb](https://www.themoviedb.org) in your API Settings of your account. If you donâ€™t already have an account, you will need to create one in order to request an API Key.

Add your API Key in the `gradle.properties` file in your project like this:

```
API_KEY="YOUR_API_KEY"
```

More info about the TMDb API [here](https://developers.themoviedb.org/3/getting-started/introduction) 

### Prerequisites

  * TMDb API Key
